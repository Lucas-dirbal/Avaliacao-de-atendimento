# Avaliacao de Atendimento

Projeto simples com:

- uma pagina inicial para escolher o atendente
- uma pagina para cada atendente: Lucas, Nicolas, Leandro, Pedro e Willian
- um `db.json` compartilhado como base de dados
- um dashboard geral lendo o mesmo JSON

## Como rodar

1. Tenha o Node.js instalado.
2. Rode `npm start`.
3. Abra `http://127.0.0.1:3000`.

## Arquivos principais

- `server.js`: servidor HTTP e API
- `db.json`: banco de dados em JSON
- `index.html`: menu inicial
- `dashboard.html`: dashboard geral
- `lucas.html`, `nicolas.html`, `leandro.html`, `pedro.html`, `willian.html`: telas dos atendentes
